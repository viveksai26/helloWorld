# Hello World Java Project

A simple Java project that prints "Hello, World!" to the console.

## Project Structure
```
src/
  └── HelloWorld.java
```

## How to Compile and Run

### Compile
```bash
javac src/HelloWorld.java
```

### Run
```bash
java -cp src HelloWorld
```

## Requirements
- Java Development Kit (JDK) 8 or higher

## Output
```
Hello, World!
```
